🎯 시그널 시스템

✓ 그룹 분류 (외국계/기관/대형기관/개인)
50개 거래원 → 4개 그룹.
대형기관 8개 (NH/KB/한투/삼성/한화/미래/신한/하나) 별도 추적
✓ 거래량 대비 비율 (%) 정규화
시가총액·거래량 무관 일관 임계값. 5일/20일 윈도우.
✓ 추세 국면 분류 (4단계)
상승 / 분배의심 / 하락 / 중립. MA20 + MA60 + 60일 신고가 기준.
✓ 베어리시 다이버전스
신고가권 + 스마트머니 음수 + 5일 가속화. 가장 강한 매도 시그널.
✓ 분배 패턴 (개미↔대형기관)
개미 매수 + 대형기관 매도 동시 → 점수 +3~5 (지속성 가중)
✓ 외국계 컨센서스 매수
5일 외국계 매수자수 - 매도자수 ≥ 8 + 외인 5일 비율 양수
✓ 거래원 집중도 (HHI + Top Share)
HHI ≥ 1500 또는 단일 거래원 5일 평균 점유율 ≥ 35% → 변동성 위험
✓ 거래량 다이버전스
가격 5일 +5% but 거래량 비율 < 0.8 → 약한 상승, 탑 형성 가능
✓ 트레일링 스탑
60일 신고가 -10% 이탈 시 추세 깨짐 컨펌
✓ 시너지 (Cluster Score)
3+ 카테고리 동시 발동 시 +2~3 보너스
✓ OBV / CMF / MFI
가격-거래량 다이버전스 자동 탐지 (외부 API 불필요)
✓ 행동 권고 4단계
전량매도 / 매도 / 부분익절 / HOLD / 신규매수
