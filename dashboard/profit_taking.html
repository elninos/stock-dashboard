🎯 익절 룰 (Profit-Taking Rules)

"평가이익이 많다"는 함정. 매도하지 않으면 시장 변동에 그대로 노출됩니다.
이익을 단계적으로 실현하면서 일부는 추세에 태우는 것이 익절 시스템의 목표.
RULE 1 · 도달 기반 분할 익절
💰 +50% / +100% / +200% 도달 시 1/4씩 익절
이익이 클수록 변동성도 큼 → 단계별로 잠금. 마지막 1/4은 추세에 태움.
실행: 한 번에 다 팔지 말고 4번에 나눠서. "원금 + α 회수" 다음에 추세 보유.
RULE 2 · 분배 시그널 강제 익절
🚨 다이버전스 / 분배 패턴 / 거래량 다이버전스 발동 시 1/3 익절
가격은 신고가지만 큰손이 빠지는 중 = 곧 빠질 가능성 큼. 즉시 부분 익절.
지표: OBV 분배 다이버전스, MFI ≥ 80, CMF ≤ -0.10, 외국계+기관 20일 음전환
RULE 3 · 트레일링 스탑 (이익 잠금)
🔻 60일 신고가 -10% 이탈 시 즉시 1/2~전량 익절
추세 깨짐 컨펌. 평가이익이 크다면 주가가 -10% 빠질 때 절대 금액 손실이 큼.
예시: +200% 종목이 -10% 빠지면 평가이익이 +170%로 줄어듬 (체감 큼)
RULE 4 · HOLD 조건
🟢 외국계 컨센서스 매수 + 신고가권 유지 + 분배 시그널 없음
"절대 안 판다"가 아니라 "지금은 안 판다". 추세가 살아있는 동안만 보유.
매일 시그널 재평가하고, RULE 2/3 발동 즉시 분할 익절로 전환.
