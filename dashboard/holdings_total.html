통합 데이터 소스:
• KIS API: 거래원/투자자별 매매 (실시간 수급)
• 패턴 분류: 거래량/변동성/추세 → 분배/펌프/잔여 자동 구분
• 브리핑 (briefing_summary.json): 텔레그램 9채널 + 블로그 5개 sentiment
• 뉴스 (stock_news.json): 종목별 호재/악재
• DART 공시: 임원 매수/매도, 5%주주 변동, 자사주 취득/처분 (180일)

특징:
• 수급 vs 펀더멘털 충돌 시 (예: SK하이닉스) 둘 다 표시 — "관찰" 등급
• 패턴이 "펌프"면 트레일링 스탑 우선
• "잔여" 패턴 + 매수 시그널 → 저점 매수 후보
