🎯 분석 목적

거래원 데이터의 한계 인정: 강세장에선 노이즈, 38일치는 너무 짧음.
대신 OHLCV 자체에 답이 있음. 큰손이 빠질 때 가격+거래량 패턴에 직접 흔적 남김.

탐지하는 7가지 신호

📉 1. 분배일 (Distribution Day)
음봉 -0.5% 이상 + 거래량이 평소(20일 평균)의 1.5배 이상.
기관/외국인이 매도하는 직접 흔적. O'Neil 방식 4주 누적 5+ 시 시장 상투.
⬆️ 2. 위꼬리 긴 봉 (Long Upper Wick)
위꼬리가 전체 range의 50% 이상. 신고가 도달했지만 매도 압력에 밀림.
❌ 3. Failed Breakout (가중 +2)
전일 20일 신고가 갱신 + 다음날 -3% 이상 하락.
"신고가 함정". 매수 세력이 매도 세력을 못 이김. 가장 강력한 사전 경고.
💥 4. Volume Climax (가중 +2)
거래량 평소의 3배 이상 + 음봉 또는 위꼬리.
분배의 절정 — 큰손이 마지막 매물 정리.
⬇️ 5. 갭 하락
시가가 전일 종가 -2% 이상. 야간/장전 정보로 인한 매도 결정.
🌊 6. 변동성 음봉 (Wide Range Down)
range가 평소의 1.8배 + 음봉 -2% 이상. 패닉 매도 가능성.

💡 종합 결론

이 신호들의 가치:
• 거래원 데이터 없이도 OHLCV만으로 작동 → 모든 종목에 즉시 적용
• 강세장 노이즈 적음 → 큰 추세는 안 끊고, 진짜 분배만 잡음
• Lead time 명확히 측정 가능 → 며칠 전에 잡았는지 검증됨

실전 룰 제안:
1. Failed Breakout: 무조건 1/3 즉시 익절 (가장 강한 단일 신호)
2. Volume Climax: 1/3 익절 + 트레일링 스탑 -10% 강화
3. 분배일 4주 누적 5건+: 1/3 익절 + 추가 매수 금지
4. 위꼬리 긴 봉이 신고가에서 발생: 즉시 1/4 익절 + 모니터링
