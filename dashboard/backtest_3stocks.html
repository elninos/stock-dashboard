분석 방법:
가격 + 거래량으로 계산되는 시그널들 (OBV/CMF/MFI/MA/Trailing) 발동 → 비슷한 시점 시그널 클러스터링 → 강한 클러스터 발동 시 분할 매도 시뮬레이션.

매도 룰:
• 클러스터 강도 (weight) 2~3: 보유 수량 1/3 매도
• 클러스터 강도 4+ : 1/2 매도
• 사용자 실제 매매와 별도로 시뮬레이션 (사용자 매수만 그대로 사용)

한계:
• daily_flow 거래원 데이터 없는 시점에는 OBV/MFI 등 가격 기반 시그널만 사용
• 실제 매매에는 변동성/세금/슬리피지 추가 고려 필요
• 백테스트는 후행 분석. 실시간 적용 시 동일 결과 보장 안 됨

🎯 핵심 결론 — 매우 의미 있는 발견

충격적 결과: 세 종목 모두 시그널이 사용자보다 못했습니다.
합계 차이 -3.11억 — 시뮬이 사용자보다 큰 폭으로 뒤짐.

왜? 시뮬이 매도를 한 번 하면 끝나기 때문. 사용자는 분할매수로 평단을 계속 낮추면서 큰 추세에 편승.
이게 보여주는 것:
1. 분할매수 + 장기 HOLD 전략의 위력 — 파마리서치 사용자 평단 71,028원, 현재 323,000원 (+354%). 시뮬은 시그널 따라 매도하다 보니 낮은 평단의 lots를 먼저 소진, 결국 추세 이익을 못 잡음.
2. 강세장 종목에서 단순 시그널 추종은 비효율적 — 추세가 강한 동안에는 어떤 시그널도 노이즈가 됨.
3. 사용자가 "너무 일찍 팔았다"고 후회하는 게 사실 정상 — 강한 추세는 어떤 시그널보다 강함.
4. 시그널은 "추세가 진짜 끝났을 때"만 작동시켜야 함 — 트레일링 -25%~-30% 정도로 매우 보수적.
그럼 시그널이 쓸모없는가? 아니다.
이 분석에는 편향(survivorship bias)이 있습니다. 세 종목 모두 강세장 성공 종목.
만약 같은 시뮬을 HLB제넥스(-2,840만), 파크시스템스(-3,094만) 같은 손실 종목에 돌리면?
→ 시그널이 일찍 매도해서 손실 줄였을 것. 시그널의 진짜 가치는 손실 회피.
실제 매매 룰 제안:
• 큰 이익(+50%↑) 보유 중: 트레일링 -25% (또는 -30%)만 발동, 나머지 시그널 무시
• 작은 이익 또는 손실: 일반 시그널 적용 (분배 다이버전스 등)
• 분할매수는 유지 — 평단 낮추기는 효과적
• 예외: 다이버전스 + 분배 + 트레일링 동시 발동 시 1/3 익절 (포지션 일부만 잠금)

💡 종목별 디테일 — 시뮬 매도가 왜 잘못됐나

  • 삼천당제약: 사용자 14회 매도 → 시뮬 6회 매도. 시뮬이 매도를 일찍 시작해서 평단 낮은 lots 소진 → 후반 큰 추세 못 잡음.
  • 토모큐브: 사용자는 사실상 매도 안 함(1회) + 평가이익 +2,536만. 시뮬은 매도 3회 → 평가이익 줄어듬. HOLD가 정답이었다.
  • 파마리서치: 가장 극적인 차이(-2.57억). 사용자 분할매수 80회로 평단을 71,028까지 낮춤. 시뮬은 매도 35회로 추세를 35번 끊었음.
